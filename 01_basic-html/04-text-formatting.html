We can format this text a lot of basic ways: bold (using the <strong> tag), italic (or emphasis, using the <em> tag), and strikethrough (using the <strike> tag).

We can also created pre-formatted (also called monospaced) text using the <pre> tag: 

  This text will appear as a different font, with each character the same width. This is also great for ASCII art!  Note: no automatic line-breaks when the page is resized...
Like <pre>, use <code> for monospaced text within a sentence.

Sometimes you want more than one space between letters or words. One might think you could just type multiple spaces - nope! Instead, you must use a 'special character' in your HTML. Spaces are written as <&nbsp;> in your HTML source code. Producing this result:

This t e x t has     extra s  p  a  c  e  s inserted.
