We can format this text a lot of basic ways: bold (<strong>), italic (or emphasis, <em>), and strikethrough (<strike>).

We can also created pre-formatted (also called monospaced) text using the <pre> tag: 

    This text will appear as a different font, with each character the same width. This is also great for ASCII art!  Note: no automatic line-breaks when the page is resized...

Notice the space between paragraphs? We can change that later, but using a line-break rather than a new paragraph is also useful, making text that looks like this:
This is text is the result of a line-break!

You can also add a "line break opportunity" where the browser will break if it needs to using the 'wbr' tag.

The tag for a line-break is <br />... notice the / inside the tag; this means that it doesn't require a closing tag (images are similar).
